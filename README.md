**Amorce** is a universal, zero-friction project scaffolder and workflow manager.

Stop wasting hours configuring linters, formatting tools, Git hooks, and environment standards. `amorce` bootstraps your project structure and strictly enforces enterprise-grade quality from commit zero, allowing you to focus on what truly matters: writing code.

## 🎯 Philosophy

1. **Zero-Friction Setup**: Answer a few interactive prompts, and your entire environment is ready.
2. **Enterprise Standards by Default**: Whether you write Rust, Node.js, or React, `amorce` applies the most robust community standards.
3. **Workflow Guardian**: Integrated Git hook management ensures that no bad commit, secret leak, or unformatted code ever makes it to your repository.

## 🇫🇷 Proudly Crafted in France

"Amorce" (ah-mohrs) is a French word meaning *primer* or *catalyst*. It represents the initial spark that sets a powerful process in motion. This tool is designed with the rigor and elegance of French software engineering, aiming for reliability and developer comfort.
